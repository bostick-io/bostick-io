## What I'm Building

### ClockPoint — ServiceNow Workforce Wellness App
A ServiceNow-native application that reframes time tracking as a driver of employee wellbeing and team performance. Built with a four-person team during the RiseUp with ServiceNow Veterans Cohort (2026).

Three pillars: compliance (clock-in, breaks, shift reports), health (automated movement prompts with smart skip guards), and culture (buddy-pairing at every wellness check). The headline feature is an AI Assistant on the manager dashboard — plain-English questions about team wellness, answered by Claude via the Anthropic API using a curated data payload pattern.

**Stack:** ServiceNow Service Portal, GlideRecord, Flow Designer, Scheduled Jobs, Script Includes, Chart.js, Anthropic Claude API

[View the ClockPoint repo](https://github.com/bostick-io/clockpoint)
